### Gwent Draw Simulator

궨트의 시작과 끝은 핸드관리라고 생각하여 멀리건 연습용으로 만듦

약간만 손보면 하스스톤이나 리그오브룬테라용으로 사용가능

### Download

  1. bin 에서 exe파일과 dll 파일을 다운받아 한 폴더에 저장한다.
  2. deck.txt를 다운받아 자신이 원하는 덱으로 수정한다.
  3. 프로그램 실행 후 덱 불러오기해서 사용한다.
